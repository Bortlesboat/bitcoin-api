Quick Start

No signup, no API key. Hit these endpoints right now.

Base URL

All endpoints are prefixed with:

https://bitcoinsapi.com/api/v1

All requests are GET (read-only) unless noted. All responses are JSON with a {data, meta} envelope.

Try it now

Check if the API is alive
curl https://bitcoinsapi.com/api/v1/health
Get fee recommendations (save money on your next transaction)
curl https://bitcoinsapi.com/api/v1/fees/recommended
Explore the genesis block
curl https://bitcoinsapi.com/api/v1/blocks/0
Check the mempool
curl https://bitcoinsapi.com/api/v1/mempool

Authentication

Most GET endpoints work without any authentication. Register for free to unlock advanced features.

How it works

Pass your API key in the X-API-Key header:

curl -H "X-API-Key: your_key_here" https://bitcoinsapi.com/api/v1/mining/hashrate

No key? You still get access to all core endpoints. Keys unlock higher rate limits, streaming, and advanced analytics.

Tier Auth Rate Limit Block Range Access
Anonymous None 200 req/min Last 144 blocks Core GET endpoints
Free X-API-Key 500 req/min Last 144 blocks + Mining, stats, POST
Pro X-API-Key 2,000 req/min Last 1,008 blocks + Streaming, analytics
Enterprise X-API-Key 5,000 req/min Last 2,016 blocks + Priority support

Register for a free key at /pricing. Self-hosted instances have no rate limits.

Fees (7 endpoints)
GET /api/v1/fees/recommended

Response Format

Every response follows a consistent envelope pattern.

Success Response

{
  "data": {
    "status": "healthy",
    "block_height": 886421
  },
  "meta": {
    "timestamp": "2026-03-09T14:30:00Z",
    "cached": true,
    "cache_age_seconds": 4,
    "request_id": "a1b2c3d4"
  }
}

Error Response

{
  "error": {
    "code": 404,
    "message": "Block not found",
    "request_id": "e5f6g7h8"
  }
}

data contains the endpoint-specific payload. meta includes the UTC timestamp, cache status, and a unique request ID for debugging. Error responses replace both with a single error object.

Error Codes

Standard HTTP status codes with descriptive messages.

Code Meaning Common Cause
400 Bad Request Invalid parameter (e.g., negative block height, malformed txid)
403 Forbidden Endpoint requires API key or higher tier
404 Not Found Block, transaction, or address not found on chain
422 Validation Error Missing required parameter or wrong type
429 Rate Limited Too many requests. Check Retry-After header.
500 Server Error Internal error. Include request_id when reporting.
502 Node Unavailable Bitcoin Core node is unreachable or circuit breaker tripped

Rate Limits

Sliding window rate limiting with daily caps. Self-hosted instances have no limits.

Rate limits are applied per API key (or per IP for anonymous requests). When you hit a limit, the API returns 429 with a Retry-After header indicating seconds until your next allowed request.

Response headers on every request:

X-RateLimit-Limit: 60 X-RateLimit-Remaining: 58 X-RateLimit-Reset: 1741529520

Need higher limits? Upgrade your tier or self-host with zero rate limits.
