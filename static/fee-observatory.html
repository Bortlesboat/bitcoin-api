Bitcoin Fee Observatory

Live comparison of fee estimators — see who's accurate and who's overpaying

The Fee Observatory dashboard is not currently running.

Start it with: cd bitcoin-fee-observatory && streamlit run app.py --server.port 8505

In the meantime, use the API endpoints below:

Scoreboard

Per-source accuracy ranking and overpayment stats

GET /api/v1/fees/observatory/scoreboard

Block Stats

Per-block fee percentiles and min/max feerate

GET /api/v1/fees/observatory/block-stats

Estimates

Multi-source fee estimate time series

GET /api/v1/fees/observatory/estimates
