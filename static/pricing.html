Frequently Asked Questions

Is Satoshi API really free?

Yes, the hosted API and self-hosted versions are both free. The hosted API gives you 1,000 requests/day without any signup, or 10,000 requests/day with a free API key. The self-hosted version is Apache-2.0 licensed with unlimited requests — run it on your own infrastructure with zero cost.

What happens if I exceed rate limits?

You'll receive HTTP 429 responses with a Retry-After header indicating how long to wait before making another request. Rate limits reset on a sliding window basis.

Can I switch between tiers?

Yes, upgrade or downgrade anytime. Your API key stays the same — only the rate limits and daily caps change.

Do you offer annual pricing?

Not currently. Pro is billed monthly via Stripe. Cancel anytime with no long-term commitment.
