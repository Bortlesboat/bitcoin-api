About Satoshi API

Bitcoin fee intelligence for AI agents.

Why We Built This

Satoshi API was built because Bitcoin Core's JSON-RPC wasn't designed for the AI agent era. As AI agents increasingly need real-time financial data, there was no Bitcoin API with native MCP integration, fee intelligence, or agent-friendly response formats. Satoshi API fills that gap.

What We Do

Bitcoin fee intelligence that saves money on every transaction. We wrap a Bitcoin Core full node with analyzed, structured data — fee recommendations, mempool analysis, block statistics, and real-time streams — all accessible via REST API or MCP tool calls.

  • Fee intelligence: "Send now or wait?" recommendations that save real money
  • Mempool analysis: Congestion levels, fee buckets, next-block predictions
  • Real-time streams: SSE and WebSocket feeds for blocks, fees, and whale transactions
  • MCP integration: 43 tools, 6 prompts, and 7 resources for any MCP-compatible agent
  • Agent-first design: Consistent JSON envelopes, deterministic pagination, self-serve registration

Open Source

Apache-2.0 licensed. Full source code on GitHub. Self-hostable on your own node with zero vendor lock-in. Run pip install satoshi-api and point it at your Bitcoin Core instance.
